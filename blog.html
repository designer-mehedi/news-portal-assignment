    Var, Let, Const

    Variables declared with var is globally scoped. Variable that is declared with var can be accessed outside of scope. But var is functional scoped. Variables can not be accessed from outside whenever the variable is declared with var inside a function. Let is block scoped that means any variable that declared with let can not be accessible outside a block. Variable values can be reassigned using let. The variable values can be reassigned or reset as new values and can be changed or modified old values. Const is also block scoped, but the variable values cannot be reassigned again that means whenever a variable value is set by const cannot be reassigned again. But the values can be modified or changed old values without reassigning a new value.

    Arrow function is a ES6 feature. It’s also called as fat arrow function. Regular and arrow functions are work similar in manner but have some certain differences. Both the functions give us the same output, but arrow function uses fewer lines unlike regular function. Curly brackets are not necessary if one expression is present. Parentheses are also not required if just one argument is passed in a single parameter. Return keyword is not needed in a simple single line of code.
